# Data Directory

This directory should contain processed data files:

```
data/
├── nicer/           # NICER folded lightcurves
├── xmm/             # XMM folded lightcurves
├── nustar/          # NuSTAR folded lightcurves
├── combined/        # Combined/merged data
└── simulation/      # Simulation outputs
```

Data files are not included in the repository due to size.
Contact the author for access to processed data.
